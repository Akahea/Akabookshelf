# BookShelf

A modern full-stack library management application built with React, TypeScript and Node.js.

## Features

-  Manage your personal library
-  Rate your books
-  Search and filter
-  Reading statistics
-  AI-powered recommendations
-  User accounts

## Tech Stack

- React 19
- TypeScript
- Express
- PostgreSQL
- Prisma
- Tailwind CSS
